📊 Prediqt-App Project Overview
Prediqt is a stock prediction web app designed to take in stock tickers and return predictions powered by multiple AI-driven trainer models, combined into a single ensemble forecast. It includes a web-based frontend, a Python backend (FastAPI), a library of machine learning models, and reporting visualizations.

🧠 Core Idea
Users input a stock ticker

A fleet of trainer models run to analyze different types of market data

Trainers generate predictions stored in logs

A fusion script combines all trainer outputs into one final prediction

Accuracy reports are generated from historical results

A frontend dashboard displays all this in a clean interface

🔁 Typical Workflow (Full Stack)
User Input (Frontend):
From index.html, a user enters a stock ticker.
→ script.js sends request to backend (api.js)

Backend API (FastAPI):
main.py receives the request and runs trainers (or loads logs)
→ Calls predictor.py and fusion_fuser.py or trainer_fusion2.py

Model Execution:
Each trainer (1–14) runs and logs to predictive_logs/
→ trainer_fusion2.py fuses their results

Results Rendered (Frontend):
Fused prediction sent back to frontend
→ script.js updates charts, scores, accuracy panels

Accuracy Reports:
generate_accuracy_report.py and others generate .json logs
→ accuracy_report.js and merged_report.js render them into charts in merged_accuracy.html

🧠 Tips for Future You
If predictions aren’t updating, check the logs in /predictive_logs/ and make sure the auto trainers are running

To manually run all trainers:
python run_all.py

To regenerate accuracy pages:
python generate_accuracy_report.py && python generate_accuracy_merged.py

Use train_predictor2.py if train_predictor.py isn’t behaving (or vice versa)

If frontend JS seems outdated, check if you're reading from old .json files

🔁 Want to Keep Trainers Running?
Use auto_trainer.py with APScheduler to loop through all trainers every hour

Or create a FastAPI route that runs them on demand

Or run run_all.py as a cron job every N minutes

🔍 To Debug
Log not updating? → Check auto_trainer.py and file permissions

Frontend stale? → Clear browser cache or rebuild Cordova wrapper

Backend down? → Run uvicorn main:app --reload to relaunch API
